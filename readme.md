# Netflix Analytics Project

## Project Overview

This project analyzes Netflix content using:

- SQL
- SQLite
- Python
- Pandas
- Matplotlib

## Dataset

Netflix dataset containing:

- Title
- Director
- Country
- Release Year
- Rating
- Duration
- Genre

## SQL Concepts Used

- GROUP BY
- COUNT()
- Window Functions
- ROW_NUMBER()
- RANK()
- PARTITION BY
- Subqueries

## Analysis Performed

1. Content Growth Over Time
2. Rating Distribution
3. Country Analysis
4. Genre Analysis
5. Director Analysis

- Python
